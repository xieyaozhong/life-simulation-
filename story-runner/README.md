# 無盡故事運行器

位於原專案的 `story-runner/` 子目錄，不修改根目錄的人生模擬器。

## 功能

- 開始／停止持續生成文字劇情
- 四段生成速度
- 隨機人物、職業、性格、目標與秘密
- 隨機物品、地點、事件與世界氣氛
- 世界狀態與故事線記憶
- 人物、物品或事件重複交會後，建立「故事入口」
- 點擊入口，以關聯角色的第一人稱視角閱讀支線
- 可將選定支線帶回主世界繼續發展
- 使用 `localStorage` 自動保存世界狀態
- 手機與桌面響應式介面
