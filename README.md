# 人生模擬器 Life Path Simulator

這是一個不需要安裝套件、可離線使用的單頁網頁應用程式。

## 使用方式

1. 解壓縮資料夾。
2. 直接雙擊 `index.html`。
3. 輸入角色名稱與人生主題。
4. 完成七個人生選擇。
5. 查看自動生成的履歷、學經歷與作品 Demo。
6. 可使用「列印 / 存成 PDF」或「匯出資料」。

## 功能

- 七階段人生選擇
- 五項能力值成長
- 自動生成學習歷程與工作經歷
- 自動生成履歷摘要、技能標籤、人生關鍵字
- 自動生成三個作品集 Demo
- JSON 資料匯出
- 瀏覽器列印與 PDF 輸出
- 手機與桌機響應式版面

## 檔案

- `index.html`：主要介面
- `style.css`：視覺與響應式設計
- `app.js`：遊戲邏輯、履歷與作品生成規則
